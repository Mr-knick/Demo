Install/compile:

Open visual studio 2019. 
Create a standard console application. 
Attach or paste over program.cs with contents of this program.
Build the program.
By defult the program will be located in the ~\bin\Debug\netcoreapp3.1\*" directory. Recommend building in release mode and using its equivalent directory. 
Log and database files are built by running the program. 

Running the program:
All instructions for running are given as prompts within the program. On first run the database and database password are generated. To reset from installation just delete the database.db file located in same directory as the .exe
